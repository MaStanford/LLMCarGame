# Game Revamp Roadmap

## Phase 1: Bug Fixes

- [x] XP not granted for enemy kills
- [x] KillCountObjective.kill_count never incremented
- [x] Difficulty modifiers not applied to enemies
- [x] Flamethrower has no collision detection
- [x] Level stat bonus uses wrong constant (0.1 instead of 0.05, max 100 instead of 50)

## Phase 2: Equipment & Loot

- [x] Equipment slot system (armor plating, engine mods, tires, etc.)
- [x] Tiered loot drops (Common / Uncommon / Rare / Legendary)
- [x] Equipment stat modifiers (speed, durability, damage, fuel efficiency)
- [x] Shop inventory overhaul with equipment sales
- [x] Salvage system (break down loot into crafting materials)
- [ ] Visual equipment display on car

## Phase 3: Story & Quests

- [ ] Main story quest chain per faction
- [ ] Branching quest dialog with reputation consequences
- [ ] Named NPC quest givers in cities
- [ ] Escort / delivery / investigation quest types
- [ ] Narrative events triggered by world exploration
- [ ] Lore collectibles scattered in the wasteland

## Phase 4: Bosses & Endgame

- [ ] Unique boss encounter per faction territory
- [ ] Boss arenas with environmental hazards
- [ ] Boss-specific loot tables (legendary weapons/equipment)
- [ ] Final boss unlocked after faction storylines complete
- [ ] New Game+ with scaled difficulty and retained equipment
- [ ] Endgame challenge zones with elite enemies
